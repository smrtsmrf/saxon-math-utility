Getting Started

  1. Create Student Accounts: To create student accounts, click on the Create Student Accounts link on the left sidebar. You may create up to 3 accounts at a time. The idea is that there will be a generic account for all algebra users, one for all geometry users, and one for all algebra 2 users. Though if you need to create more users you are welcome to do so. Usernames must be unique, so consider using your school name as part of the username. If "alg", "geo", or "alg2" is part of the username, then students will be directed to the appropriate homework page after login.
  2. Modify HW: Only administrator accounts are authorized to make changes to the homework, but teachers may submit modification requests. To access the modification page, click on the title on the homework page.
    • Administrators: On the modifications page, click on the lesson number you want to skip, turning the button grey. When you have finished selecting which lessons you want to skip, click the "update" button at the bottom of the page, and it will redirect you to the newly modified homework.
    • Teachers: Teachers follow the same process to modify the homework, but instead of the modification being made, they are prompted to either submit an update request, or enter an admin key.
      • Submit a Request: Teachers explain their reasoning, and upon submission, an email is sent to the administrator with the request and reasoning. This email contains a link for administrators to make the change themselves, or send an admin key back to the requesting teacher.
      • Enter an Admin Key: If the administrator chooses to send an admin key, an email is generated and sent to the requesting teacher. The teacher then can copy and paste this code into the request box (after clicking the request update button and selecting "enter admin key") and then the request will be implemented.
  3. Reset Homework: There are two places that you may reset the homework - on the modifications page, and on the "Reset HW" part of the admin page.
    • Modifications page: A reset button appears underneath the subject you are currently viewing. Clicking this will go back to not skipping any lessons in this subject. For teachers, this button works differently: clicking it will go back to not making any changes to the current homework.
    • Admin page: Clicking the "Reset HW" link on the left sidebar will give you the option to reset ALL subjects back to not skipping any lessons.
  4. View/Delete Users: Clicking on the "Users" link on the left sidebar will allow you to view all users associated with your school (admins, teachers, and students). You may also delete users from this page. Be careful, as this cannot be undone.
  5. Forgot password: At this time there is no "password reset" or "forgot username/password" feature. Your email address can only be associated with one account, so please write down your username/password in a secure location.
